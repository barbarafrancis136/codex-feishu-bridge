## 🎯 核心策略

**所有报告采用"永久保留，仅最新版本"策略：**

```
每次生成新报告时：
1. 生成新报告（带当天日期）
2. 查找所有同类报告
3. 按修改时间排序
4. 删除所有旧报告
5. 仅保留最新版本
```

## 🛠️ 自动清理脚本

### 基金新闻日报 (Python)

```python
def cleanup_old_reports(report_dir):
    """清理旧报告，仅保留最新版本"""
    import glob
    files = glob.glob(f"{report_dir}/fund-daily-brief-*.md")
    if len(files) > 1:
        # 按修改时间排序，删除旧的，保留最新
        files.sort(key=os.path.getmtime, reverse=True)
        for old_file in files[1:]:
            os.remove(old_file)
```

### 基金关联工作流 (Bash)

```bash
# 获取所有报告文件，按时间排序，删除旧的保留最新
cd "$REPORT_DIR" && ls -t fund-workflow-*.md 2>/dev/null | tail -n +2 | xargs rm -f
```
